# 英语题库系统

一个基于Node.js的英语题库网站，支持用户注册、题目发布、在线答题和排名功能。

## 功能特点

1. **用户管理**
   - 用户注册（仅需用户名和密码）
   - 用户登录/注销
   - 个人资料查看
   - 密码修改

2. **题目管理**
   - 发布题目（支持Markdown文档上传和直接输入）
   - 图片上传
   - 题目类型设置（选择题/填空题）
   - 答案解析

3. **答题功能**
   - 查看题目列表
   - 在线答题
   - 即时反馈答题结果
   - 查看答案解析

4. **排名系统**
   - 上传题目数量排名
   - 答题正确率排名

5. **管理员功能**
   - 查看所有题目
   - 审核并删除题目
   - 需提供删除原因

## 技术栈

- **后端**: Node.js, Express
- **数据库**: SQLite
- **模板引擎**: EJS
- **文件上传**: Multer
- **密码加密**: bcryptjs
- **Markdown解析**: marked
- **HTML净化**: DOMPurify
